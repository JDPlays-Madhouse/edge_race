# Edge Race

[![Latest Published to Factorio Mods](https://github.com/JDPlays-Madhouse/edge_race/actions/workflows/auto_publish.yml/badge.svg)](https://github.com/JDPlays-Madhouse/edge_race/actions/workflows/auto_publish.yml)

Documentation: [GitHub Wiki](https://github.com/JDPlays-Madhouse/edge_race/wiki)
Mod Page: [Factorio Mods](https://mods.factorio.com/mod/edge_race)

This limits you do expand in one direction by limiting your width/height to a ringworld
like setting and adds void behind spawn to force growth in 1 direction only.

**THE FACTORY MUST GROW** will you survive.

Use the mod settings to adjust the settings first, read the manual ([wiki](https://github.com/JDPlays-Madhouse/edge_race/wiki)
) to know what they change.
